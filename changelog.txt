Version: 1.1.2
Date: 17. 04. 2026
  Features:
    - Added a new Vehicles & Infrastructure crafting group to reduce clutter in the default logistics tab.
    - Moved vehicles, rail infrastructure, power distribution, floors, foundations, landfill, scaffolding, and platform items into the new infrastructure group.
    - Added prototype identity polish so overlapping planet unlocks feel more distinct instead of looking like duplicate rewards.
    - Gave Moshine big accumulators a clearer Grid Reserve identity with high storage and slower charge/discharge behavior.
    - Gave Moshine and Rubia long inserter lines different roles: Moshine favors awkward long-reach data-center logistics, while Rubia is framed around wind-safe hostile-surface logistics.
    - Clarified Cubium dream belt recipes as cube-loop conversion routes rather than new belt tiers.
    - Clarified Ribbonia lane splitters as compact lane-control tools rather than another belt speed upgrade.
    - Clarified Muluna silicon solar panels as an alternate supply-chain recipe, while Ribbonia/Tiny Solar compact panels are treated as fragile one-tile layout tools.

  Changes:
    - Removed the K2SO Singularity Tech Card ingredient from the Deep Space Tech Card recipe so singularity science stays in the final transceiver chain instead of appearing too early.
    - Cleaned up Muluna green plastic research so it no longer pulls in confusing unrelated planet requirements.
    - Added Rubia's long stack inserter dependency on Moshine's normal long stack inserter when both technologies exist.
    - Added Imersite asteroid spawns to late Beetlejuice and Deep Space routes so Imersite is no longer effectively locked around Fulgora.
    - Kept Terra Palus as a required content mod while leaving its route alone, since it is handled as a moon of Gleba by its source mod.

  Bugfixes:
    - Integrated the Maraxsis/K2SO collision-mask drill compatibility behavior directly into Razi Protocol and marked the standalone fix-fix mod as incompatible.
    - Fixed late-game Deep Space card recipe clutter caused by the Singularity Tech Card appearing before the intended finale.
    - Added final-fixes guards around several optional prototype polish changes so missing optional content does not crash data stage.

  Info:
    - Updated README and mod page description to document the new crafting tab, prototype identity polish, Imersite asteroid route changes, and current compatibility notes.
